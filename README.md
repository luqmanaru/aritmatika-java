# aritmatika-java
Program sederhana yang mendemonstrasikan penggunaan operator aritmatika dasar dalam Java.

---

## Fitur

- Menerima input dua angka dari pengguna
- Melakukan operasi pengurangan
- Menampilkan hasil perhitungan

---

## Contoh Output

```
Input angka-1: 3
Input angka-2: 4
Hasil = -1
```
